## What this is

This is a private, encrypted record of my bank accounts, investments, and insurance
policies. It exists so you are not left guessing or searching when something happens
to me. It does **not** replace my will or wasiat — it is a practical directory to sit
alongside it.

## Steps to access the live app (if you want to browse everything)

1. Open: **https://estate-vault-fiza.web.app**
2. Sign in with the registered email — contact [Lawyer / trusted technical contact]
   if you need the sign-in credentials reset.
3. When asked for the **Master Password**, enter exactly what was given to you by
   the lawyer (case-sensitive, no extra spaces).
4. You'll see the dashboard: Bank Accounts, Investments, Insurance & Takaful,
   Trusted Contacts, Documents Locator, and Wasiat Notes.

## If the app itself isn't reachable

If a check-in was missed past the grace period, you'll receive an email with a file attached:
**estate-vault-backup.html**. This is all you need:
1. Save the attachment to your computer or phone.
2. Open it in any web browser (Chrome, Safari, Edge) — works fully offline, no internet needed.
3. Type in the Master Password when prompted.

That's it — no sign-in, no visiting any website, no app. The file itself contains everything
needed to decrypt and display the information once the Master Password is entered.

## When next-of-kin get emailed automatically

If check-ins are missed past the grace period, the trusted contacts listed in Settings receive
the file above automatically. Otherwise, nothing is sent on any regular schedule — you'd only
see this if something has actually gone unusually quiet.
